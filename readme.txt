Add the Follow Button to your website to increase engagement and create a lasting connection with your audience.

== Description ==

* Add Twitter Follow Button to your wordpress blog.
* Display buttons in different languages
* Very simple configuration.

Visit plugin site:
http://www.darkomitrovic.com/wp-plugin/twitter-follow-button/


= Languages = 

* English
* French
* German
* Italian
* Japanese
* Korean
* Russian
* Spanish
* Turkish


== Installation ==

1. Unzip the file
2. Upload it to `/wp-content/plugins/` directory
3. Activate it from the plugins section
